# Pinang-Performa-Data-Engineering
Merupakan produk pinjaman berbasis digital proses yang fokus pada pembiayaan sektor usaha produktif. Bank Raya mendukung dalam memajukan pekerja informal atau gig worker, memberikan layanan Pinang Performa untuk meningkatkan produktifitas &amp; kapasitas gig worker dalam berusaha melalui Pinjaman Digital.
